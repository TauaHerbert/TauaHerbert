No meu dia a dia, presto suporte técnico aos usuários, solucionando chamados de infraestrutura e manutenção de equipamentos. Em paralelo, dedico meus estudos ao desenvolvimento de software, construindo soluções com Python e Java. Estou constantemente atualizando meus conhecimentos e buscando novos desafios na área de tecnologia. Meu objetivo é aprender e aplicar essa base técnica para criar soluções inovadoras e gerar impacto real.
